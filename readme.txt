This plugin allows you to highlight text on a post or a page then launch a search for the selected text on the website without leaving the page.

== Description ==

[Maneno - Search selected text](http://www.sourcecreativity.com/) is a great time saver because you don't need to copy and paste your text in to the search box, hit search and change the page to view the results. 
Just select text inside your post or page an icon will be displayed click on it a nice popup will be displayed with all results in the same page. 
You can choose the number of displayed posts and click on the "Load more results" button to load more results with a nice effect.

= Features =

* Specify the number of posts displayed per page
* Specify the number of characters per post
* Allows you to change the search icon trough dropdown list
* CSS file can be easily accessed and modified

== Installation ==

1. Download and unzip the plugin.
2. Upload the `maneno` directory to `/wp-content/plugins/`.
3. Activate the plugin through the 'Plugins' menu in WordPress.
